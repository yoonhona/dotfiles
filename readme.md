윈도우에서 개발하다 맥북으로 개발 한 뒤에는 운영체제가 느려지거나 꼬이는 상황이 발생하여 포맷 후 운영체제를 재설치하는 일이 거의 발생하지 않았다.

그렇지만 아예 재 설치하는 일이 없는 것도 아니고 재 설치를 진행하면 거의 꼬박 하루를 잡아야 하기에

나만의 개발환경을 자동화해본다.

사전 작업
* xcode 설치 필요
* brew mas로 AppStore 어플을 설치하기 때문에 AppStore 로그인 필요
